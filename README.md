# Hotel Management System

## Description

The Hotel Management System is a console-based C++ project that helps manage hotel operations such as room booking, food ordering, stock management, and daily sales tracking. The system updates room and food availability automatically and calculates the total collection for the day.

---

## Features

* Room booking system
* Food ordering management
* Stock availability checking
* Sales and collection report
* User-friendly console interface

---

## Technologies Used

* C++
* Console-based interface

---

* Variables and Data Types

* Loops
* Switch Case
* Functions
* Input and Output Handling
## How to Run the Project
1. Open the project in any C++ compiler or IDE.
2. Compile the program.
3. Run the executable file.
4. Choose options from the menu to manage hotel operations.
## Menu Options
1. Book Rooms
2. Order Pasta
3. Order Burger
4. Order Noodles
5. Order Shake
6. Show Sales & Collection
7. Exit
